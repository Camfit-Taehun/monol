# UC-07: 개발 코치 모드(반복 패턴 피드백)

## 가치(Why)
개인 반복 실수/개선 포인트를 자동 피드백해 역량을 향상합니다.

## 범위(What)
- 트리거: `monol coach daily`
- 산출: weekly/daily 리포트 markdown

## 선행 조건/의존(Task)
- TASK-200 events

## 구현 설계/개발 플랜(How)
### 1) 지표
- 실패 테스트 Top N
- 룰 위반 Top N
- PR 리뷰 코멘트 유형(선택)

### 2) 리포트 생성
- `.monol/reports/coach-daily.md` 저장
- events에 `coach.report.generated`

## 수정/생성 파일(초안)
- `monol/src/core/coach/report.ts`
- `monol/tests/coach-report.test.ts`

## 테스트 시나리오(재현 절차)
### 테스트
- fixture events로 리포트 생성 스냅샷 테스트
- 데이터 부족 시 안내 메시지
