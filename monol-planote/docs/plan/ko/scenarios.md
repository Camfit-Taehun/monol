# Planote — 시나리오 (KO)

이 문서는 반드시 검증해야 하는 해피패스 및 엣지/예외 케이스를 나열한다.

핵심 시나리오:
- S-001 init/serve
- S-002 탐색/검색
- S-003 섹션 수정요청 주석
- S-004 선택 주석 + 재매칭
- S-005 사이클 생성 + 번들 생성
- S-006 에이전트 실행 + 리비전 감지 + 해결
- S-007 파일명 변경(주석 생존)
- S-008 헤딩 삭제(obsolete)
- S-009 비UTF-8 파일
- S-010 XSS 시도
- S-011 동시 서버 락
- S-012 대규모 플랜 성능

상세 단계/기대결과는 이전 초안의 시나리오 문서를 회귀 기준으로 유지한다.
